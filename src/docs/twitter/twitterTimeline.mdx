# TwitterTimeline

Display a TwitterTimeline by including the component in your `.mdx`. The `username` prop is required.

<Props of={TwitterTimeline} />

## Basic usage

<Playground>
  <TwitterTimeline username="PaulieScanlon" height="400px" />
</Playground>

## Width

<Playground>
  <TwitterTimeline username="PaulieScanlon" height="400px" width="100%" />
</Playground>

## Theme

<Playground>
  <TwitterTimeline username="PaulieScanlon" theme="dark" height="400px" />
</Playground>

## Show Likes

<Playground>
  <TwitterTimeline username="PaulieScanlon" showLikes={true} height="400px" />
</Playground>
