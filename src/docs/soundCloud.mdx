# SoundCloud

Display a SoundCloud player by including the component in your `.mdx`. The `soundCloudLink` prop is required.

<Props of={SoundCloud} />

## Basic usage

<Playground>
  <SoundCloud soundCloudLink="tracks/499997862" />
</Playground>

## Color

<Playground>
  <SoundCloud soundCloudLink="tracks/188189839" color="e23f88" />
</Playground>

## Visual

<Playground>
  <SoundCloud
    soundCloudLink="tracks/196287837"
    visual={true}
    width="300px"
    height="300px"
  />
</Playground>
